AWS Static Website Hosting

Static website hosting is a feature that you can use to host your static website on Amazon S3. On a static website, individual webpages include static content. They might also contain client-side scripts. By contrast, a dynamic website relies on server-side processing, including server-side scripts such as PHP, JSP, or ASP.NET. Amazon S3 does not support server-side scripting.

For more information, please visit Amazon S3 Website Hosting.
